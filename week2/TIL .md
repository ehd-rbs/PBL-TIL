### 2. 핵심 정리 (내 언어로)

- 반응형 레이아웃
모바일   -> @media (max-width: 767px)
태블릿   -> @media (min-width: 768px) and (max-width: 1023px)
데스크톱 -> @media (min-width: 1024px)

- 배지 표시
  울타리 -> .profile-container { position: relative; }
  스티커 -> .badge { position: absolute; }   -- 배지를 오버레이 상태로 만듬

### 4. 느낀 점

화면 크기에 따라 카드 개수가 자동으로 조절되는 것을 보며, 하나의 코드로 모든 기기에 대응하는 유연함의 중요성을 느낌.
복잡한 계산이 없어도 그리드 설정만으로 깔끔한 정렬이 완성되는 과정이 매우 흥미로웠음.
relative 와 absolute를 활용해 배지를 이미지 위에 올리면서 정확한 기준점 설정의 중요성을 깨달음.
작은 스타일 차이가 페이지 전체의 완성도를 결정한다는 점이 수행과정에서 가장 인상깊었음.
