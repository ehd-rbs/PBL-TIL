# 📘 Today I Learned

### 1. 오늘 배운 내용
- Supabase 연동

### 2. 핵심 정리 (내 언어로)

Supabase 연결을 통해 mock 데이터 대신 실제 DB 데이터 조회
로그인한 사용자만 데이터를 추가/삭제할 수 있도록 구현
회원가입, 로그인, 로그아웃 및 로그인 상태 유지 기능 구현
Supabase URL과 anon key는 .env.local에 저장하고 git에 올리지 않는다.

### 4. 느낀 점

이번 과제를 통해 Supabase를 이용하면 프론트엔드에서도 실제 DB와 인증 기능을 연결할 수 있다는 점을 알게 되었다.
특히 mock 데이터가 아닌 실제 데이터를 조회하고, 로그인 여부에 따라 기능을 제한하는 과정이 중요하다고 느꼈다.
환경변수와 타입 관리를 통해 보안성과 안정성을 함께 고려해야 한다는 점도 배웠다.
